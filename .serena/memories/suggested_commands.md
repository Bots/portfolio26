# JPportfolio тАФ Suggested Commands

## Package management
- `pnpm install` тАФ install dependencies
- `pnpm postinstall` тАФ run after install (nuxt prepare)

## Development
- `pnpm dev` тАФ start dev server on http://localhost:3000
- `pnpm build` тАФ build for production
- `pnpm generate` тАФ static generation
- `pnpm preview` тАФ preview production build locally

## Code quality
- `pnpm lint` тАФ run Biome check (`biome check .`)
- `pnpm format` тАФ run Biome format write (`biome format --write .`)

## System utilities (Linux)
- `git` тАФ version control
- `pnpm` тАФ package manager (all commands use pnpm, not npm/yarn)
